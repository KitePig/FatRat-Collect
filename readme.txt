= 胖鼠留 =
* 有用的不爽的地方, 接受大家提出的合理的需求, 去迭代.
* 看到这里有没有觉得很牛皮. 哈哈, 牛皮还是要吹的!

* 胖鼠是完全内嵌到wordpress插件里面,不需要脱离wordpress,
* 不会安装其他任何第三方软件. 功能强大。相比其他插件. 胖鼠很有优势.
* 胖鼠是使用 Html,Jquery语法 爬取和剔除数据. 比其他采集器, 更灵活,
* 胖鼠软件的设计思想分三大块. ①爬虫模块, ②配置模块, ③数据模块
* ①爬虫模块主要使用配置模块的各种特色配置来爬取数据.
* ②配置模块为爬虫模块提供支持.
* ③数据模块支撑胖鼠各种特色发布功能!

安装：
1. 在插件中搜索 <strong>胖鼠采集</strong> 安装即可
2. 将插件文件上传到/wp-content/plugins/目录 即可
